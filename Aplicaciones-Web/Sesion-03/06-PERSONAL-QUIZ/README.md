# 🎮 Personal Mini Quiz

> Developing an interactive personal quiz using JavaScript to evaluate user answers and display the final score without reloading the webpage.

---

# 📖 Overview

In this activity, I developed a **Personal Mini Quiz**, an interactive web application where visitors answer multiple-choice questions about me and instantly receive their score.

The quiz was built using HTML, CSS, and JavaScript. Users select one answer for each question, and JavaScript compares the selected answers with the correct ones. After submitting the quiz, the application calculates the total number of correct answers and displays the final result dynamically without refreshing the page.

This project combines many of the JavaScript concepts learned throughout Session 03, including event handling, objects, DOM manipulation, conditional statements, and dynamic content updates.

# ⚙️ How It Works

The quiz follows these steps:

1. The visitor answers each multiple-choice question.
2. JavaScript waits until the **"View Results"** button is pressed.
3. The selected answer for every question is obtained.
4. Each answer is compared with the correct answer.
5. Every correct answer increases the score.
6. The total score is calculated.
7. The final result is displayed instantly without reloading the page.
