Mis Proyectos

Durante mi formación universitaria he desarrollado diferentes proyectos relacionados con el desarrollo de software, aplicaciones web y programación.

Proyecto 1 - UniTrack

Sistema web para el control de asistencia de estudiantes y docentes. Incluye autenticación de usuarios, generación de códigos QR, reportes y administración de información académica.

Proyecto 2 - Learn How To Code

Repositorio educativo donde documento mi proceso de aprendizaje en programación utilizando C#, Kotlin, estructuras de datos, algoritmos y desarrollo de proyectos prácticos.

Proyecto 3 - Desarrollo Web

Desarrollo de un sitio web personal utilizando HTML5 y CSS3 para presentar información académica, proyectos personales y medios de contacto siguiendo buenas prácticas de desarrollo web.
