# 📬 Contact Page

## 📖 Overview

In this activity, I created a new web page named **contacto.html** as part of my personal website. The page allows visitors to send messages through a contact form while maintaining the same visual design used throughout the rest of the website.

The page uses the external stylesheet **estilos.css**, ensuring a consistent appearance across all pages.

---

## 🎯 Objectives

- Create a new HTML page.
- Connect the page to the shared stylesheet.
- Build a contact form using semantic HTML.
- Practice HTML forms.
- Improve website navigation.
