# 🎨 Introduction to CSS3

## 📖 Overview

In this activity, I learned how to use CSS3 to improve the visual appearance of a personal website created with HTML5. Instead of placing styles inside each HTML file, I used an external stylesheet to apply consistent formatting across the entire website.

The project demonstrates the separation between structure (HTML) and presentation (CSS), making the website easier to maintain, update, and expand.

---

## 🎯 Objectives

- Learn the purpose of CSS3.
- Connect an external stylesheet to multiple HTML pages.
- Apply typography and color styles.
- Customize headings and paragraphs.
- Style hyperlinks.
- Design a navigation bar.
- Improve the appearance of images.
- Create responsive content containers.
