# 📖 Overview

Welcome to **Session 01** of the **Web Applications** course.

This session introduces the fundamental concepts of web development and the technologies used to build websites. It explains how the Internet and the World Wide Web work, the Client–Server model, Front-end and Back-end development, HTTP and HTTPS, and the purpose of web hosting.

As the practical activity, I created my first HTML5 webpage from scratch using Visual Studio Code.
