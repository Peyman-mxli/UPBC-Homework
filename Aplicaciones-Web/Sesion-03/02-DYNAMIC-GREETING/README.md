# 🌞 Dynamic Greeting

> Creating a personalized greeting that changes automatically based on the current time using JavaScript.

---

# 📖 Overview

In this activity, I developed my first interactive JavaScript feature by displaying a dynamic greeting on the homepage of my personal website.

Using the **JavaScript Date API**, the program determines the current system time and displays an appropriate greeting such as **Good Morning**, **Good Afternoon**, or **Good Evening**. This exercise demonstrates how JavaScript can manipulate HTML elements dynamically and respond to real-world data.

---

# 🎯 Learning Objectives

During this activity I learned how to:

- Use JavaScript variables
- Obtain the current system date and time
- Extract the current hour
- Apply conditional statements
- Manipulate HTML elements dynamically
- Display different content depending on the time of day

# ⚙️ How It Works

The application follows these steps:

1. Creates a new **Date** object.
2. Retrieves the current hour using `getHours()`.
3. Evaluates the hour using `if / else if / else`.
4. Selects the correct greeting.
5. Updates the HTML element using `textContent`.
