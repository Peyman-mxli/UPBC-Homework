# 🔝 Scroll To Top Button

> Creating a floating button that improves website navigation by allowing users to smoothly return to the top of the page.

---

# 📖 Overview

In this activity, I implemented a **Scroll To Top** button using JavaScript and CSS.

The button remains hidden while the page is at the top. As soon as the user scrolls down, the button appears in the bottom-right corner of the screen. When clicked, it smoothly scrolls the page back to the top, providing a better browsing experience.

This feature is commonly found on modern websites with long pages and demonstrates how JavaScript can react to browser events and manipulate CSS dynamically.

# ⚙️ How It Works

The application performs the following steps:

1. The page loads with the button hidden.
2. JavaScript listens for the **scroll** event.
3. When the user scrolls more than 300 pixels, the button becomes visible.
4. If the user scrolls back to the top, the button disappears.
5. Clicking the button smoothly scrolls the page back to the beginning.
