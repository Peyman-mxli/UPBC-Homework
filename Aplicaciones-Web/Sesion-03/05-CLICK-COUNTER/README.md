# ❤️ Click Counter

> Creating an interactive button that counts user clicks in real time using JavaScript.

---

# 📖 Overview

In this activity, I created a simple but interactive **Click Counter** using JavaScript.

Each time the visitor clicks the button, the counter increases immediately without reloading the webpage. This exercise demonstrates how JavaScript can listen to user events, update variables, and modify HTML content dynamically.

Although simple, this project introduces one of the most important programming concepts: **application state**, where a variable stores information that changes while the application is running.

# ⚙️ How It Works

The application performs the following steps:

1. A variable stores the number of clicks.
2. JavaScript waits for the user to click the button.
3. Every click increases the counter by one.
4. The updated value is displayed instantly on the webpage.

The counter resets whenever the page is refreshed.
